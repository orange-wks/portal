水木とClaudeが使えなかった

Anthropic がクォータ騒動の対応で揺れていた週、水曜・木曜と Claude がまともに使えない状態になった。「#ClaudeQuotaGate」のタグとともに制限到達のスクショが大量に流れていた。Claude Code にはプロンプトキャッシュが正常に機能しないバグもあり、意図せず大量トークンを消費させていたらしい。

代替を考えた。Gemini と Codex は有料プランがあるが、どちらも使いたくない。Gemini は挙動が不安定で、Codex（GPT系）は口調が嫌いだ。質問に対して過集中で答えてくる。大局観がなく、仕事のパートナーにしたくない。

禁断症状的に、木曜のうちに Alibaba Cloud（Qwen API）のアカウントを登録した。すぐ使えるわけではなく、身分証明（免許証かパスポート）の審査に2日ほどかかる。先手を打った。

詫び石 $200 が届いた

木曜に登録したら、Anthropic から補償として $200 のクレジットが届いた。「これがあれば Alibaba 要らんかも」と思った。

土曜12時頃、$200 あるしとぶん回したら、1時間強で $70 消費していた。

これはダメだと判断した。Claude MAX $200 プランだけで生きていくのはどうせ無理。Alibaba に行くことにした。

Alibaba Cloud で Qwen を使う

ホビー用途なら中華系でも構わない。性能に興味もある。GLM など他にも選択肢はあるが、よく知らないのでメジャーな Qwen にした。

安い。qwen3.6-plus で入力 $0.5/1M・出力 $3/1M（従量課金）。256k トークンを超えると入力 $2/1M・出力 $6/1M に跳ね上がるが、それでも Claude と比べると30分の1以下だ。無料枠もあるが、1M トークンは57分で溶けた。

審査が通れば API キーを発行できる。Claude Code のバックエンドとして呼び出す設定も通った（LiteLLM 経由でプロキシが必要だが、動いた）。

コーディングプランの契約方法がわからない

ここで詰まっている。

Alibaba Cloud には「コーディングプラン」的なものがあるらしいのだが、どこから契約するのかがわからない。DashScope のコンソールを探しても見当たらない。

なお現時点の請求額は $1.47 だ。キャッシュが効いているのか、無料枠なのか、まだ反映されていないだけなのか、よくわからない。このあとどうなるかドキドキしている。
